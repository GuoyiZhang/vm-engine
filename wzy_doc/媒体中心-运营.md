##运营登录
<span id = "login"></span>
###接口：
	/**
     * 运营登录
     *
     * @param dataString
     * @return
     */
	String login(String dataString) 
									 
###请求示例:
+ dataString :

		{
			"username":"wzy01",
			"password":"W_Z_y",
			"isRem":0
		}
###响应：
		{
		    "code":0,
		    "data":{
		        "account":{
		            "username":"wzy",
		            "id":123
		        }
		    },
		    "msg":"Success"
		}
